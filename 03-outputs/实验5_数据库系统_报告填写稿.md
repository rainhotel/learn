#### ③ 分页查询：查询学生选课情况第 3 页，每页 4 条

SQL语句（文本）：

方法一：`OFFSET ... FETCH NEXT`

```sql
DECLARE @pageIndex INT = 3;
DECLARE @pageSize INT = 4;

SELECT *
FROM StuCourse
ORDER BY 学号, 课程号
OFFSET ((@pageIndex - 1) * @pageSize) ROWS
FETCH NEXT @pageSize ROWS ONLY;
```

方法二：`ROW_NUMBER()`

```sql
DECLARE @pageIndex INT = 3;
DECLARE @pageSize INT = 4;

SELECT TOP (@pageSize) *
FROM (
    SELECT ROW_NUMBER() OVER (ORDER BY 学号, 课程号) AS rownumber, *
    FROM StuCourse
) AS temp_row
WHERE rownumber > ((@pageIndex - 1) * @pageSize)
ORDER BY rownumber;
```

#### ② while 语句

SQL语句（文本）：

```sql
BEGIN TRAN;

DECLARE @sumcredit INT;

SELECT @sumcredit = SUM(学分)
FROM Course;

WHILE @sumcredit < 50
BEGIN
    UPDATE Course
    SET 学分 = 学分 + 1;

    SELECT @sumcredit = SUM(学分)
    FROM Course;
END

SELECT *
FROM Course;

PRINT N'当前总学分为：' + CAST(@sumcredit AS VARCHAR(20));

ROLLBACK TRAN;
```

执行前表数据（截图）：

- 截 `Course` 全表。

执行结果（截图）：

- 当前初始总学分是 `21`，循环会执行。
- 最终总学分会达到不小于 `50`。
- 因为使用了事务并回滚，不会永久修改数据库。

#### ① 标量函数：根据学号返回某学生总分

SQL语句（文本）：

```sql

    DROP FUNCTION dbo.fn_total_grade;
GO

CREATE FUNCTION dbo.fn_total_grade(@stu_no CHAR(6))
RETURNS INT
AS
BEGIN
    DECLARE @total INT;

    SELECT @total = SUM(CAST(成绩 AS INT))
    FROM StuCourse
    WHERE 学号 = @stu_no;
```

#### ③ 多语句表值函数：查询某专业学生平均成绩

SQL语句（文本）：

```sql
IF OBJECT_ID('dbo.fn_major_avg', 'TF') IS NOT NULL
    DROP FUNCTION dbo.fn_major_avg;
GO

CREATE FUNCTION dbo.fn_major_avg(@major VARCHAR(50))
RETURNS @result TABLE
(
    专业名 VARCHAR(50),
    平均成绩 DECIMAL(5,2)
)
AS
BEGIN
    INSERT INTO @result
    SELECT s.专业名, AVG(CAST(sc.成绩 AS DECIMAL(5,2)))
    FROM Student s
    JOIN StuCourse sc ON s.学号 = sc.学号
    WHERE s.专业名 = @major
    GROUP BY s.专业名;

    RETURN;
END;
GO

SELECT *
FROM dbo.fn_major_avg(N'计算机科学与技术');
```

  - `平均成绩 = 81.44`

```sql
    DROP PROCEDURE dbo.proc_course_avg;
GO

CREATE PROCEDURE dbo.proc_course_avg
    @cno CHAR(4),
    @c_avg DECIMAL(5,2) OUTPUT
AS
BEGIN
    SELECT @c_avg = AVG(CAST(成绩 AS DECIMAL(5,2)))

GO
```

### （2）定义触发器 Ctrig：课程学分不能小于 2

① 定义触发器的 SQL 语句（文本）：

```sql
IF OBJECT_ID('dbo.Ctrig', 'TR') IS NOT NULL
    DROP TRIGGER dbo.Ctrig;
GO

CREATE TRIGGER dbo.Ctrig
ON Course
AFTER INSERT, UPDATE
AS
BEGIN
    SET NOCOUNT ON;

    UPDATE c
    SET 学分 = 2
    FROM Course c
    JOIN inserted i ON c.课程号 = i.课程号
    WHERE c.学分 < 2;
END;
GO
```

② 自行设计数据更新操作来验证触发器，具体 SQL 语句（文本）：

```sql
BEGIN TRAN;

UPDATE Course
SET 学分 = 1
WHERE 课程号 = '2001';

SELECT *
FROM Course
WHERE 课程号 = '2001';

ROLLBACK TRAN;
```

③ 数据更新操作的执行结果（截图，包括执行前相关数据的情况）：

- 如果触发器生效，则更新后查询时 `课程号='2001'` 的 `学分` 应自动变为 `2`，而不是 `1`。
