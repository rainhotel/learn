# Otsu 阈值分割实验结果

## 任务

- 采用 Otsu 法对 `Fig1046(a)(septagon_noisy_shaded).tif` 和 `Fig1040(a)(large_septagon_gaussian_noise_mean_0_std_50_added).tif` 进行阈值分割。
- 对比分割前后的图像效果。

## 阈值结果

| 图像 | Otsu 阈值 | 归一化阈值 |
| --- | ---: | ---: |
| `Fig1040(a)` | 118 | 0.4627 |
| `Fig1046(a)` | 49 | 0.1922 |

## 结果分析

### Fig1040(a)

- 原图中的七边形整体明显亮于背景，但叠加了较强高斯噪声。
- Otsu 自动给出的阈值为 `118`，可以较完整地保留七边形主体轮廓。
- 分割后主体区域基本被提取出来，但前景内部和背景区域仍可见较多散点噪声。
- 这说明 Otsu 作为全局单阈值方法能够完成主体分离，但对强噪声不具备抑制能力。

### Fig1046(a)

- 原图中的七边形与背景灰度接近，同时存在明显阴影和背景亮度渐变。
- Otsu 自动给出的阈值为 `49`，主体轮廓能够被清晰分离。
- 分割后左侧和主体内部较干净，但右侧受背景渐变影响，仍有较多误分的白色噪点。
- 这说明当图像存在光照不均或阴影时，全局 Otsu 依然能提取主要目标，但局部背景变化会降低分割纯净度。

## 结论

- 对这两幅图像，Otsu 法都能较好地提取七边形主体，说明该方法对“目标和背景仍有整体灰度差异”的场景是有效的。
- `Fig1040(a)` 的主要问题是噪声导致前景和背景都出现散点。
- `Fig1046(a)` 的主要问题是阴影和亮度不均导致右侧背景误分更明显。
- 如果要继续提升结果，可在阈值分割前增加平滑滤波，或改用局部阈值分割方法。
