# 图像直方图处理实验导学

## Start Here

- 这次实验最重要的不是把函数名背下来，而是看懂“灰度分布怎么变成处理结果”。
- 如果你现在时间不多，先完成 `histogram_lab.m` 里的实验 1 和实验 2，因为这两部分最容易拉开报告质量。
- 下次打开时优先看这 3 个文件：`README.md`、`projects.md`、`histogram_lab.m`。

## Why This Matters

- 直方图是最基础的图像统计描述之一，后面的阈值分割、增强、对比度调整都离不开它。
- 这次实验把“图像分析”和“算法实现”连在一起，既能帮你过实验，也是在给后续数字图像处理打地基。
- 如果能把人工阈值、Otsu、均衡化和 CLAHE 的差别讲明白，这份报告就已经不只是“跑结果”了。

## Fast Path

1. 把实验图像放到 Matlab 当前目录，或者修改 `histogram_lab.m` 里的 `imageDir`
2. 先运行实验 1，观察三幅图的直方图，微调人工阈值
3. 再运行实验 2，记下 `graythresh` 输出的阈值
4. 完成实验 3，理解你手写的映射公式
5. 再运行实验 4 和 5，保存截图
6. 最后按“原理、代码、结果、分析、总结”写报告

## Common Traps

- `graythresh` 返回的是归一化阈值，不是 0 到 255 的整数。
- 人工阈值不一定和 Otsu 完全一致，只要你能解释为什么不同就可以。
- 直方图均衡化后并不保证直方图完全平坦，只是趋向更均匀。
- 自适应直方图均衡化更适合局部亮度不均的图像，但也可能放大噪声。
