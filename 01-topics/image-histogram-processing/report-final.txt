一、实验目的
1. 了解直方图的性质和应用，理解图像二值化的原理并掌握其算法实现。
2. 掌握直方图均衡化、规定化算法的原理及实现。
3. 了解自适应直方图均衡化算法及其应用。

二、实验内容与结果分析
1. 利用直方图特性人工选择阈值进行二值化
首先读取 eight.tif、rice.png 和 Fig0338(a)(blurry_moon) 三幅图像，并同屏显示原图、原图直方图和二值化结果。根据直方图分布与视觉效果，人工阈值分别选为：eight.tif 为 0.500，rice.png 为 0.350，Fig0338(a)(blurry_moon) 为 0.450。
从结果可见，eight.tif 能较好分离硬币和背景，但硬币内部仍有少量噪点；rice.png 能分出大部分米粒，但背景噪声较多；blurry_moon 图像边缘模糊，人工阈值选择主观性较强。实验说明：人工阈值法直观易用，但对经验依赖明显，鲁棒性一般。

2. graythresh 函数与 Otsu 阈值对比
graythresh(I) 基于 Otsu 最大类间方差法，输出 [0,1] 的归一化阈值。实验得到三幅图像 Otsu 阈值为：eight.tif 为 0.649，rice.png 为 0.514，Fig0338(a)(blurry_moon) 为 0.275。
与人工阈值比较如下：
图像名称：eight.tif；人工阈值：0.500；Otsu阈值：0.649；结果对比：Otsu 结果更干净，硬币边界更完整。
图像名称：rice.png；人工阈值：0.350；Otsu阈值：0.514；结果对比：Otsu 分割更准确，背景噪声更少。
图像名称：Fig0338(a)(blurry_moon)；人工阈值：0.450；Otsu阈值：0.275；结果对比：Otsu 对月亮主体保留更稳定。
综合来看，Otsu 法自动化程度高、稳定性好，在本实验中整体优于人工阈值法。

3. Matlab 手写实现直方图均衡化（不用 histeq）
使用 pout.tif，先统计直方图并归一化，再计算累计分布函数 CDF，最后根据映射表完成灰度变换。处理后图像对比度明显增强，人物头发、衣物褶皱和背景纹理更清晰；后直方图分布范围更宽，灰度利用更充分。
说明手写均衡化实现正确，能够提升整体对比度。但均衡化后也可能放大局部噪声，视觉风格会发生变化。

4. 直方图规定化（目标直方图为 Fig0809(a)）
使用 histeq(I, hgram) 以 Fig0809(a) 的直方图作为目标分布，对 pout.tif 进行规定化。结果显示：规定化后图像灰度层次明显改变，风格向目标分布靠拢；后直方图总体趋势接近目标直方图，但不可能完全一致。
这表明规定化是“按目标分布做映射”，不同于均衡化“追求更均匀分布”的目的。

5. 自适应直方图均衡化（CLAHE）
查询 Matlab 函数 adapthisteq 并进行实验。相较全局均衡化，CLAHE 能更好增强局部细节。实验中人物头发、衣服纹理、背景网格细节均更清晰；后直方图更平滑，灰度覆盖更广。由于使用限制对比度机制，噪声增强可控。
结论：CLAHE 更适合亮度不均、局部细节不足的图像增强场景。

三、关键算法代码
1. 人工阈值与 Otsu 二值化
I = im2double(imread('rice.png'));
manualThreshold = 0.35;
BW_manual = imbinarize(I, manualThreshold);
level = graythresh(I);
BW_otsu = imbinarize(I, level);

2. 手写直方图均衡化
function [J, map, cdfValues] = my_histeq_gray(I)
    Iu = im2uint8(I);
    counts = imhist(Iu, 256);
    cdfValues = cumsum(counts) / numel(Iu);
    map = uint8(round(255 * cdfValues));
    J = map(double(Iu) + 1);
end

3. 直方图规定化
