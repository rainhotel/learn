## 二、实验原理

图像直方图是图像灰度分布情况的统计描述，它反映了各灰度级像素出现的频数或频率。通过观察直方图，可以初步判断图像的整体亮度、对比度以及目标与背景之间的可分离程度，因此直方图分析是图像分割和图像增强的重要基础。

图像二值化是将灰度图像中的像素按某一阈值分为前景和背景两类的过程。当像素灰度值大于阈值时归为一类，否则归为另一类。人工阈值法需要结合原图和直方图手动选取阈值，优点是直观，缺点是依赖经验。Otsu 方法是一种经典的自动阈值选择方法，其基本思想是在所有可能阈值中寻找一个能够使前景与背景类间方差最大的阈值。Matlab 中的 `graythresh` 函数正是基于 Otsu 算法实现的，返回值是 0 到 1 之间的归一化阈值，可直接与 `imbinarize` 配合使用。

直方图均衡化是通过对图像灰度值进行非线性变换，使输出图像的灰度分布尽可能均匀，从而提升图像整体对比度。其核心是先统计原图像的灰度直方图，再计算累计分布函数（CDF），最后依据累计概率构造灰度映射关系。直方图规定化与均衡化不同，均衡化的目标是让分布更均匀，而规定化的目标是让输出图像的灰度分布尽可能接近给定目标直方图，因此具有更强的针对性。

自适应直方图均衡化是将整幅图像划分为若干局部区域，分别在局部区域内进行直方图增强，再将结果拼接起来。相比全局均衡化，它更适合处理亮度不均匀、局部细节较弱的图像。为了避免局部噪声被过度放大，Matlab 中通常使用限制对比度的自适应直方图均衡化函数 `adapthisteq`，即 CLAHE 方法。

## 三、实验内容与结果分析

### 1. 人工选择阈值对图像进行二值化

首先读取 `eight.tif`、`rice.png` 和 `Fig0338(a)(blurry_moon)` 三幅灰度图像，并绘制原图直方图。在观察图像灰度分布后，分别为三幅图像手动设定阈值：`eight.tif` 为 0.500，`rice.png` 为 0.350，`Fig0338(a)(blurry_moon)` 为 0.450。随后使用 `imbinarize` 函数对图像进行二值化处理，并同屏显示原图、原图直方图和处理结果。

从实验结果来看，`eight.tif` 的背景较亮、硬币区域较暗，人工阈值为 0.500 时已经能够较好地区分目标与背景，但硬币内部仍存在少量亮斑。`rice.png` 的背景纹理较复杂，阈值为 0.350 时能够分离出大部分米粒，但背景中仍保留了较多噪声。`Fig0338(a)(blurry_moon)` 由于图像边缘模糊且灰度过渡较为平滑，人工选择阈值时更依赖主观经验，阈值为 0.450 时可以较清晰地保留月亮主体，但局部细节仍存在缺失或粘连现象。

该实验说明，人工阈值法实现简单、直观，适合对直方图具有明显峰谷特征的图像进行快速分割，但其效果依赖实验者经验，阈值稍有变化就可能导致目标缺失或背景噪声增加。

### 2. 使用 `graythresh` 计算最佳阈值并与人工阈值比较

Matlab 中 `graythresh` 函数的基本用法为 `level = graythresh(I)`，其作用是依据 Otsu 最大类间方差法计算图像的全局最优归一化阈值。得到阈值后，可使用 `BW = imbinarize(I, level)` 完成自动二值化处理。

本实验中，三幅图像对应的 Otsu 阈值分别为：`eight.tif` 为 0.649，`rice.png` 为 0.514，`Fig0338(a)(blurry_moon)` 为 0.275。与人工阈值相比，Otsu 方法得到的阈值整体偏高或偏低，说明它是从全局统计分布出发自动寻找最优分界点，而人工方法更多依赖视觉经验和任务目标。

表1 人工阈值与 Otsu 阈值对比

| 图像名称 | 人工阈值 | Otsu阈值 | 结果对比 |
| --- | --- | --- | --- |
| eight.tif | 0.500 | 0.649 | Otsu 结果更干净，硬币边界更完整，内部亮斑更少 |
| rice.png | 0.350 | 0.514 | Otsu 结果明显优于人工阈值，米粒轮廓更清晰，背景噪声更少 |
| Fig0338(a)(blurry_moon) | 0.450 | 0.275 | Otsu 结果更稳定，月亮主体保留更完整，但局部细节仍受模糊影响 |

由图像对比可知，`eight.tif` 的直方图具有相对明显的双峰特征，因此 Otsu 能够较准确地找到前景与背景的分界点。`rice.png` 中人工阈值偏低，导致背景干扰较多，而 Otsu 阈值能更有效地分离米粒与背景。`Fig0338(a)(blurry_moon)` 的灰度分布受到大面积黑色背景影响，虽然人工阈值与 Otsu 阈值差异较大，但 Otsu 方法仍给出了较稳定的月亮主体区域。总体而言，Otsu 法具有自动化程度高、结果稳定的优点，在本实验的三幅测试图像上综合效果优于人工阈值法。

### 3. Matlab 手写实现直方图均衡化

在本实验中，使用 `pout.tif` 图像进行直方图均衡化处理，并要求不使用 Matlab 自带的 `histeq` 函数，而是自行编程实现其核心过程。具体步骤包括：先统计原图的灰度直方图，再对直方图进行归一化，计算累计分布函数，最后根据累计分布函数构造新的灰度映射表，并完成像素值替换。

从实验结果来看，均衡化后图像整体对比度明显增强，人物头发、面部阴影、衣服纹理和背景网格细节都比原图更加清晰。原图直方图主要集中在中低灰度区域，均衡化后直方图的分布范围明显拓宽，说明灰度级利用更加充分。虽然处理后直方图没有完全变成平坦分布，但整体已经明显比原图更分散，这符合离散图像直方图均衡化的实际情况。

该实验说明，直方图均衡化能够有效改善灰度集中、对比度不足的图像，但也可能带来局部噪声增强和视觉风格变化较大的副作用。因此在实际应用中，需要根据图像类型和增强目标来决定是否使用该方法。

### 4. 直方图规定化

直方图规定化实验使用 `pout.tif` 作为源图像，以 `Fig0809(a)` 的直方图作为目标分布。处理过程中，先计算参考图像的目标直方图，再调用 `histeq(I, hgram)` 使源图像的灰度分布尽可能接近目标分布。

从实验结果可以看出，规定化后的图像灰度层次发生了明显改变，整体视觉风格更接近参考图像所对应的目标直方图。与实验3中的均衡化相比，规定化的目的不是让灰度分布均匀，而是让结果图像朝着某个指定分布逼近。观察结果直方图可知，其整体变化趋势与目标直方图较为一致，但由于源图像本身的灰度结构、离散映射误差以及像素分布差异，结果直方图不可能与目标直方图完全重合。

该实验说明，直方图规定化在需要实现特定视觉风格或特定灰度分布时更有优势，它是一种“有目标”的直方图变换方法。

### 5. 自适应直方图均衡化

本实验通过 Matlab 的 `adapthisteq` 函数实现自适应直方图均衡化，并使用 `pout.tif` 作为测试图像。与全局均衡化相比，该方法在局部区域内分别进行增强，因此更适合处理亮度不均和局部细节较弱的图像。实验中设置 `ClipLimit = 0.01`，`NumTiles = [8 8]`，以实现较为平衡的局部增强效果。

实验结果表明，CLAHE 处理后图像的局部细节增强更明显，尤其是人物头发纹理、衣服褶皱以及背景细节比原图更加清晰。观察其后直方图可以看出，增强后的灰度分布更平滑，整体跨度更大，说明局部对比度得到了提升。由于采用了限制对比度机制，噪声虽然略有增强，但没有出现过度放大的情况，视觉效果较自然。

该实验说明，自适应直方图均衡化在保留图像自然感的同时，能够更有效地增强局部细节，尤其适用于光照不均匀或局部对比度不足的灰度图像。

## 四、关键算法代码

### 1. 人工阈值与 Otsu 阈值二值化

```matlab
I = im2double(imread('rice.png'));
manualThreshold = 0.35;
BW_manual = imbinarize(I, manualThreshold);

level = graythresh(I);
BW_otsu = imbinarize(I, level);
```

### 2. 手写直方图均衡化核心代码

```matlab
function [J, map, cdfValues] = my_histeq_gray(I)
    Iu = im2uint8(I);
    counts = imhist(Iu, 256);
    cdfValues = cumsum(counts) / numel(Iu);
    map = uint8(round(255 * cdfValues));
    J = map(double(Iu) + 1);
end
```

### 3. 直方图规定化核心代码

```matlab
sourceUint8 = im2uint8(I_pout);
refUint8 = im2uint8(I_ref);
targetHist = imhist(refUint8, 256);
J_spec = histeq(sourceUint8, targetHist);
```

### 4. 自适应直方图均衡化核心代码

```matlab
J_clahe = adapthisteq(sourceUint8, 'ClipLimit', 0.01, 'NumTiles', [8 8]);
```
