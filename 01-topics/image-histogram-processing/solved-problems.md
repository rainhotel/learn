# 图像直方图处理实验 Solved Problems

## Problem 1: 直方图引导的阈值分割

- Date: 2026-04-02
- Source: 数字图像处理实验任务 1 与任务 2
- Topic: 图像二值化 / 阈值选择
- Difficulty: Easy

#### Problem

- 题目原文：对 `eight.tif`、`rice.png` 和 `Fig0338(a)(blurry_moon)` 先人工选阈值二值化，再使用 `graythresh`/Otsu 计算最佳阈值并对比。

#### Final Answer

- 最终答案：
  - 人工阈值需要根据直方图谷底和目标分离效果联合确定。
  - `graythresh` 返回 Otsu 归一化阈值，可直接用于 `imbinarize`。
  - 二者不必完全一致，关键是解释各自结果为什么合理。

#### Solution

1. 先显示原图与原图直方图，观察是否存在明显双峰或谷底。
2. 先给一个人工阈值，查看二值化结果是否出现前景缺失、背景粘连或噪声过多。
3. 再调用 `graythresh` 得到自动阈值，用同样方式二值化。
4. 最后比较两组阈值和视觉效果，解释差异来自哪里。

#### Formula Or Method Used

- 公式/定理/方法：Otsu 最大类间方差法
- 含义：在所有阈值中选取使前景与背景可分性最大的那个阈值
- 适用条件：前景和背景灰度分布有一定区分度，尤其适合双峰或近双峰直方图

#### Sources

- 参考来源 1：本主题 `notes.md`
- 参考来源 2：Matlab 文档 `graythresh`

#### Knowledge Points

- 这道题对应的知识点：归一化阈值、类间方差、人工阈值与自动阈值的比较
- 这道题暴露出的薄弱点：只看数值不看图像分割质量

#### Reflection

- 这次为什么会做对/做错：是否真正根据直方图和任务目标去调阈值，而不是机械照抄数值
- 以后怎么更快识别这类题：先判断直方图峰谷结构，再决定是否适合单阈值方法

## Problem 2: 手写直方图均衡化

- Date: 2026-04-02
- Source: 数字图像处理实验任务 3
- Topic: 直方图增强
- Difficulty: Medium

#### Problem

- 题目原文：不能使用 `histeq`，自己编程实现 `pout.tif` 的直方图均衡化。

#### Final Answer

- 最终答案：
  - 先计算原图归一化直方图，再做累计求和得到 CDF。
  - 用 `s_k = round((L - 1) * CDF(k))` 构造灰度映射表。
  - 用映射表替换原像素即可得到均衡化结果。

#### Solution

1. 把灰度图转换到 8 bit 灰度级。
2. 统计每个灰度级出现次数，得到直方图。
3. 用像素总数归一化直方图，并累加得到 CDF。
4. 由 CDF 计算新的灰度映射表。
5. 用映射表查表生成输出图像。

#### Formula Or Method Used

- 公式/定理/方法：基于累计分布函数的全局灰度映射
- 含义：把原图像灰度集中区域拉开，提高整体对比度
- 适用条件：整体灰度范围偏窄、层次不明显的灰度图像

#### Sources

- 参考来源 1：本主题 `formula-sheet.md`
- 参考来源 2：本主题 `histogram_lab.m`

#### Knowledge Points

- 这道题对应的知识点：直方图、CDF、映射表、全局对比度增强
- 这道题暴露出的薄弱点：容易把“均衡化”误认为“让直方图完全平坦”

#### Reflection

- 这次为什么会做对/做错：关键在于是否真正理解了 CDF 映射而不是只会调用函数
- 以后怎么更快识别这类题：看到“增强整体对比度”“手写 `histeq`”就想到先建直方图再建映射表

## Problem 3: Otsu 法分割含噪七边形图像

- Date: 2026-06-11
- Source: 数字图像处理实验题
- Topic: 图像二值化 / Otsu 阈值分割
- Difficulty: Easy

#### Problem

- 题目原文：采用 Otsu 法对图像 `Fig1046(a)(septagon_noisy_shaded).tif` 和 `Fig1040(a)(large_septagon_gaussian_noise_mean_0_std_50_added).tif` 进行阈值分割，对比分割前后图像。

#### Final Answer

- 最终答案：
  - `Fig1040(a)` 的 Otsu 阈值为 `118`，归一化阈值约为 `0.4627`。
  - `Fig1046(a)` 的 Otsu 阈值为 `49`，归一化阈值约为 `0.1922`。
  - 两幅图都能较好分离七边形主体，但噪声和阴影会造成一定误分。

#### Solution

1. 读取两幅灰度图像并统计 0 到 255 的灰度直方图。
2. 计算每个候选阈值对应的前景、背景概率及均值。
3. 通过最大化类间方差选择最佳阈值，即 Otsu 阈值。
4. 按阈值把图像分成两类，输出二值分割图。
5. 对比原图和分割结果，分析噪声、阴影对结果的影响。

#### Formula Or Method Used

- 公式/定理/方法：Otsu 最大类间方差法
- 含义：选择能够让前景和背景可分性最大的全局阈值
- 适用条件：前景与背景存在整体灰度差异，直方图具有一定分离性

#### Sources

- 参考来源 1：本主题 `notes.md`
- 参考来源 2：输出结果 `03-outputs/image-histogram-processing/report.md`

#### Knowledge Points

- 这道题对应的知识点：全局阈值、类间方差、噪声对分割的影响、阴影对分割的影响
- 这道题暴露出的薄弱点：容易只记阈值数值，不分析误分原因

#### Reflection

- 这次为什么会做对/做错：先确认目标和背景谁更亮，再解释二值结果中的噪声来源，分析会更完整
- 以后怎么更快识别这类题：看到“单目标 + 灰度差异 + 自动阈值”时，优先考虑 Otsu；看到“阴影不均”时，警惕全局阈值局限
