# 图像直方图处理实验 Formula Sheet

## Core Formulas Or Methods

### Histogram Probability

- 名称：归一化直方图
- 表达式/方法：`p(r_k) = n_k / N`
- 含义：第 `k` 个灰度级出现的频率
- 什么时候用：分析图像灰度分布、作为均衡化和 Otsu 的基础输入
- 不该在什么时候用：需要空间位置信息时
- 常见误用：把频数和频率混为一谈

### Global Thresholding

- 名称：单阈值二值化
- 表达式/方法：`g(x, y) = 1, f(x, y) > T; 0, 其他`
- 含义：用阈值 `T` 把像素分成前景和背景
- 什么时候用：前景背景灰度差明显时
- 不该在什么时候用：光照极不均或前景背景灰度严重重叠时
- 常见误用：只看阈值数值，不看二值图实际分割效果

### Otsu Criterion

- 名称：最大类间方差
- 表达式/方法：`sigma_b^2(T) = w_0(T) * w_1(T) * (mu_0(T) - mu_1(T))^2`
- 含义：在所有阈值里选出让两类差异最大的那个
- 什么时候用：双峰或近双峰直方图、全局阈值场景
- 不该在什么时候用：局部亮度变化很大、噪声很重时
- 常见误用：把 Otsu 当成所有图像都“最优”的万能法

### Histogram Equalization

- 名称：直方图均衡化映射
- 表达式/方法：`s_k = round((L - 1) * sum(p(r_j), j = 0..k))`
- 含义：通过累计分布函数把灰度映射到更均匀的分布
- 什么时候用：整体对比度不足、灰度集中在窄区间时
- 不该在什么时候用：非常在意噪声放大或原图已经对比度很高时
- 常见误用：误以为处理后直方图必须完全平坦

### Histogram Specification

- 名称：直方图规定化
- 表达式/方法：先求源图 `T(r)`，再依据目标图 CDF 近似求 `G^-1(T(r))`
- 含义：让输出分布接近指定直方图，而不是追求均匀分布
- 什么时候用：已知理想灰度分布或要向参考图像风格靠拢时
- 不该在什么时候用：没有明确目标分布时
- 常见误用：把规定化和均衡化混成一个概念

### CLAHE

- 名称：限制对比度的自适应直方图均衡化
- 表达式/方法：局部块均衡化 + 对局部直方图裁剪
- 含义：提升局部细节，同时尽量抑制噪声被过度放大
- 什么时候用：局部亮度不均、细节埋在局部阴影中的图像
- 不该在什么时候用：图像噪声极大且不想做额外抑制时
- 常见误用：把参数调得过大导致颗粒感明显
