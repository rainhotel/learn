## Experiments

### Experiment 1: 人工阈值二值化

- 目标：根据原图直方图人工选择阈值，对三幅图像完成二值化。
- 步骤：
  - 读取 `eight.tif`、`rice.png`、`Fig0338(a)(blurry_moon)`
  - 同屏显示原图、原图直方图、二值化结果
  - 根据直方图谷底或前景背景分布调节阈值
- 结果：
  - `eight.tif` 人工阈值：`0.500`
  - `rice.png` 人工阈值：`0.350`
  - `Fig0338(a)(blurry_moon)` 人工阈值：`0.450`
- 结论：
  - `eight.tif` 和 `rice.png` 通过单阈值分割可以得到较明确的目标区域，但人工阈值对细节保留和噪声抑制较敏感。
  - `Fig0338(a)(blurry_moon)` 因边缘模糊、灰度过渡平缓，人工阈值更难一次选准，结果更依赖主观经验。

### Experiment 2: `graythresh` 与 Otsu 对比

- 目标：了解 `graythresh` 用法，并比较自动阈值与人工阈值差异。
- 步骤：
  - 用 `graythresh` 求三幅图像阈值
  - 用 `imbinarize` 完成二值化
  - 与实验 1 的人工阈值结果做对比
- 结果：
  - `eight.tif` Otsu 阈值：`0.649`
  - `rice.png` Otsu 阈值：`0.514`
  - `Fig0338(a)(blurry_moon)` Otsu 阈值：`0.275`
- 结论：
  - 与人工阈值最接近的是 `rice.png`，两者都能较好分离米粒与背景，但 Otsu 结果更干净。
  - 差异最大的图像是 `Fig0338(a)(blurry_moon)`，说明该图像灰度分布重叠较多，人工经验和类间方差准则会给出明显不同的判断。
  - 从结果看，Otsu 在三幅图像上都比人工阈值更稳定，尤其在 `rice.png` 和 `blurry_moon` 上表现更好。

### Experiment 3: 手写直方图均衡化

- 目标：不用 `histeq`，自己实现 `pout.tif` 的直方图均衡化。
- 步骤：
  - 统计原图直方图
  - 计算累计分布函数
  - 构造灰度映射表
  - 生成均衡化结果并显示前后图像与直方图
- 结果：
  - 图像视觉变化：均衡化后图像整体亮度层次更丰富，人物头发、衣服褶皱和背景纹理更清晰。
  - 直方图变化：处理后直方图分布范围明显变宽，灰度级利用更充分，整体趋于分散。
- 结论：
  - 均衡化提升了整体对比度。
  - 副作用是直方图呈离散尖峰状，局部噪声和颗粒感也会被一定程度增强。

### Experiment 4: 直方图规定化

- 目标：以 `Fig0809(a)` 的直方图作为目标分布，对 `pout.tif` 做规定化。
- 步骤：
  - 读取 `pout.tif` 与 `Fig0809(a)`
  - 用 `imhist` 生成目标直方图
  - 用 `histeq(I, hgram)` 做规定化
  - 同屏显示前后图像、前后直方图、目标直方图
- 结果：
  - 规定化后视觉效果：结果图像的灰度层次发生明显重分配，视觉风格更接近参考图像所对应的目标分布。
  - 结果直方图与目标直方图的一致程度：整体趋势较接近，但由于离散映射和源图本身限制，无法完全一致。
- 结论：
  - 规定化与均衡化相比，更像是在做“向指定灰度分布靠拢”的映射，而不是单纯提升全局对比度。

### Experiment 5: 自适应直方图均衡化

- 目标：查询 Matlab 的局部均衡化函数，并完成一次 CLAHE 处理。
- 步骤：
  - 查询 `adapthisteq` 的功能和参数
  - 读取测试图像
  - 显示处理前后的图像与直方图
- 结果：
  - 局部细节提升情况：CLAHE 处理后人物头发、衣服纹理和背景网格细节更加清晰，局部层次增强明显。
  - 噪声变化：由于采用限制对比度的方法，噪声虽然有所增强，但比普通局部均衡化更可控。
- 结论：
  - 它比全局均衡化更适合亮度分布不均、需要增强局部细节的图像。
