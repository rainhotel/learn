# Dynamic Programming Formula Sheet

## Core Formulas Or Methods

### Item 1

- 名称：DP 解题四问
- 表达式/方法：状态是什么、怎么转移、初值是什么、按什么顺序算
- 含义：这是写任何基础 DP 之前都要先回答的 4 个问题
- 什么时候用：看见“求方案数 / 求最值 / 求可行性”且存在重复子问题时
- 不该在什么时候用：题目本质是贪心、数学结论或图搜索时
- 常见误用：直接套表格，不先定义 `dp` 含义

### Item 2

- 名称：网格单向计数 DP
- 表达式/方法：`dp[cur] = 所有合法前驱状态之和`
- 含义：当前格子的方案数由所有能一步走到它的位置贡献
- 什么时候用：移动方向单调，只依赖更小坐标时
- 不该在什么时候用：存在回退、环或状态不只由坐标决定时
- 常见误用：把“从当前位置能走到哪”误写成“当前位置从哪来”

### Item 3

- 名称：P1796 马走棋转移
- 表达式/方法：`dp[i][j] = dp[i - 1][j - 2] + dp[i - 2][j - 1]`
- 含义：最后一步只能是 `(i-1,j-2) -> (i,j)` 或 `(i-2,j-1) -> (i,j)`
- 什么时候用：马只能按 `(x+1,y+2)` 或 `(x+2,y+1)` 单向前进时
- 不该在什么时候用：存在障碍、允许回走或移动规则变化时需要改写状态
- 常见误用：漏判下标合法性，或忘记设置 `dp[1][1] = 1`
