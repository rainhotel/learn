# Dynamic Programming Mastery Outline

## Phase 0: Positioning And Scope

- 先回答动态规划到底在解决什么问题
- 区分它和暴力搜索、记忆化搜索、贪心的边界
- 固定一套入门判断标准：有没有重复子问题，状态是否能递推

## Phase 1: Grid DP

- 路径计数
- 带障碍路径计数
- 最小路径和 / 最小代价
- 重点练习：状态定义、最后一步倒推、二维遍历顺序

## Phase 2: Linear DP

- 爬楼梯
- 最大子段和
- 基础 LIS / 序列递推
- 重点练习：一维状态和前缀依赖

## Phase 3: Knapsack

- 0/1 背包
- 完全背包
- 多重背包的基本思路
- 重点练习：容量维、倒序/正序遍历

## Phase 4: Structural DP

- 区间 DP
- 树形 DP
- 状态压缩 DP
- 暂不作为当前第一阶段重点

## Phase 5: Pattern Library

- 方案数类
- 最值类
- 可行性判断类
- 计数转最值 / 最值转计数的变形

## Phase Exit Criteria

### Phase 1 结束前

- 能独立写出 3 道网格 DP
- 能解释为什么遍历顺序正确
- 能快速指出起点、边界和非法状态

### Phase 2 结束前

- 能把二维问题和一维问题的状态区别讲清楚
- 能解释“最后一步”和“前缀最优”的不同思路

### Phase 3 结束前

- 能分清 0/1 背包和完全背包的遍历方向
- 能解释滚动数组为什么成立
