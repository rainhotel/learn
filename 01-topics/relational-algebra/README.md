# Relational Algebra

## Goal

- 学懂关系代数最核心的概念、运算符和解题顺序。
- 能从“公式含义”出发，而不是靠死记符号做题。

## Scope

- 包含：选择、投影、笛卡儿积、连接、除法、属性位置表示法、二维表结果书写。
- 暂不包含：SQL 优化、关系规范化、事务并发控制。

## Outcome

- 能独立解释常见关系代数运算符。
- 能看懂并手算基础关系代数题。
- 能区分“存在型”筛选和“对所有都成立”的除法语义。
- 能把中间过程写成二维表。
