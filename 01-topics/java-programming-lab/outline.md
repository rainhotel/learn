# Java Programming Lab Mastery Outline

## Phase 0: Positioning And Scope

- 明确课程实验覆盖的章节
- 区分“会运行”和“会解释”的差别
- 建立实验目录与知识目录的映射

## Phase 1: Map The Field

- Java 基础语法
- 面向对象
- 集合框架
- GUI 编程
- 异常处理
- 线程
- JSP / Servlet 入门

## Phase 2: Foundations

- 集合接口与实现类
- 迭代器与增强 for
- Swing 常见组件
- JSP 的三种基本脚本元素

## Phase 3: Core Mechanisms

- `Set` 去重与无序特性
- `Iterator` 的遍历和删除机制
- JSP 到 Servlet 的转换思想

## Phase 4: Patterns And Trade-Offs

- `HashSet` 与 `ArrayList` 的使用场景差异
- 直接写 JSP 脚本片段与 JavaBean 分层的差异
- 单线程界面与多线程界面的行为差异

## Phase 5: Deliberate Practice

- 重写教材例题
- 修改输入数据观察输出变化
- 对照运行结果写实验分析

## Phase 6: Build Something Real

- 做一个简单集合管理小程序
- 做一个带表单输入的 JSP 页面

## Phase 7: Explain, Teach, Defend

- 能讲清楚每个实验使用了哪些类、方法、接口
- 能解释实验现象和教材截图之间的对应关系

## Phase 8: Frontier And Taste

- 从 JSP 脚本页逐步迁移到 MVC 思维
- 从教材示例过渡到更规范的工程实践
