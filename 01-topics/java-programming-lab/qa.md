# Java Programming Lab QA

## 常见问题与误区

### 1. 为什么 `HashSet` 的输出顺序和教材截图不一样？

- 因为 `HashSet` 不保证元素顺序。
- 只要集合操作结果语义一致，顺序不同是正常现象。

### 2. 为什么遍历时不能直接 `vector.remove(...)`？

- 因为这会破坏迭代器的遍历状态，可能导致并发修改异常。
- 应该在遍历中使用 `Iterator.remove()`。

### 3. JSP 文件为什么不能像普通 Java 类一样直接运行？

- 因为 JSP 需要由 Web 容器先转换为 Servlet，再由容器处理请求。

### 4. `11-1` 实验最容易忽略什么？

- 输入校验。
- 如果输入为空或不是整数，页面应给出合理提示。
