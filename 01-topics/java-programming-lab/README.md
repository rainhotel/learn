# Java Programming Lab

## Goal

- 用实验驱动方式学习 Java 语言程序设计中的核心知识点。
- 不只完成作业，还要把集合、GUI、JSP、线程等内容沉淀成可复习的知识页。

## Scope

- 包含：教材例题、课程实验、代码实现、实验报告、常见错误与运行现象总结。
- 暂不包含：完整 Java Web 工程化、Spring 生态、数据库后端项目开发。

## Outcome

- 能独立完成教材中的基础实验。
- 能解释代码为什么这样写，而不只是照抄。
- 能把每次实验中的知识点整理为稳定笔记和标准题解。
