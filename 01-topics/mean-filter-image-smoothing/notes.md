# 知识笔记

## 关键规律

- 模板越大，平滑通常越强，但边缘和细节更容易丢失。
- 高斯滤波比简单算术均值更自然，因为它更重视中心邻域。
- 双边滤波同时考虑空间距离和灰度相似性，所以保边更好。
- 几何均值、谐波均值和逆谐波均值都不是“通用最优”，适用范围依赖噪声类型。

## 参数经验

- 高斯滤波中，`sigma` 过大容易让图像整体发糊。
- 逆谐波均值滤波中，`Q > 0` 更适合椒噪声，`Q < 0` 更适合盐噪声。
- 对细线条较多的图像，不宜盲目放大模板。

## 考试速记

### 滤波器用途

- 平滑滤波器：主要用于去噪，也会让图像变模糊。
- 锐化滤波器：主要用于强调边缘和细节，也可用于增强模糊图像中的细节。
- 图像增强：目标是让图像看起来更好。
- 图像复原：目标是尽量恢复图像原貌，以保真为主。

### 线性与非线性

- 线性滤波器：输出通常能表示成邻域像素的加权和，例如算术均值滤波、高斯滤波。
- 非线性滤波器：输出不是简单加权和，例如最小值滤波器、最大值滤波器、中值滤波器。

### 常见噪声类型

- 高斯噪声：灰度值围绕某个均值随机波动，是最常见的随机噪声之一。
- 脉冲噪声：常表现为盐噪声、椒噪声或椒盐噪声。
- 周期噪声：具有明显重复规律，不属于常见随机噪声。

### 常见边缘算子

- 一阶微分算子：Roberts、Prewitt、Sobel。
- 二阶微分算子：Laplacian。
- Sobel 梯度幅值若采用城区距离，计算式为 `|Gx| + |Gy|`。
- Sobel 梯度幅值若采用欧式距离，计算式为 `sqrt(Gx^2 + Gy^2)`。

### 3x3 邻域统计滤波速记

- 算术均值滤波：取邻域平均值。
- 中值滤波：将邻域内像素排序后取中间值。
- 最小值滤波：取邻域最小值。
- 最大值滤波：取邻域最大值。
- 中点滤波：`(最大值 + 最小值) / 2`。

### 边界点处理

- 如果题目没有特别说明，手算题通常只考内部像素点。
- 如果题目要求处理边界点，必须先明确边界策略，例如补零、边界复制或镜像扩展。
- 中值滤波、均值滤波等边界点的结果不是唯一的，取决于补边方式。

### 本组题易错点

- 平滑滤波和锐化滤波的作用方向相反：一个抑制细节，一个增强细节。
- 逆谐波均值滤波虽然适合脉冲噪声，但必须先区分是亮噪声还是暗噪声，再决定 `Q` 的正负。
- 几何均值滤波的平滑能力可以和算术均值滤波相比，但通常会损失更少的图像细节。
