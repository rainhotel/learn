## Why This Matters

- TCP 是 Web、分布式系统和互联网可靠传输的基础协议之一。
- 课程设计如果只写“TCP 很重要”会比较空，真正有分量的是讲清 TCP 如何随着网络环境和 Web 应用需求而演进。
- 学会这个主题后，你不仅能交这次报告，也能更容易理解 HTTP/2、负载均衡、连接池、CDN、服务治理这些后续概念。

## Mastery Path

3. 再把这些改进放到 Web 演化里理解
