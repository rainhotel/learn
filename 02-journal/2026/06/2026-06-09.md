#### 3. 分页查询：第 3 页，每页 4 条

```sql
DECLARE @pageIndex INT = 3;
DECLARE @pageSize INT = 4;

SELECT *
FROM StuCourse
ORDER BY 学号, 课程号
OFFSET ((@pageIndex - 1) * @pageSize) ROWS
FETCH NEXT @pageSize ROWS ONLY;
```

或：

```sql
DECLARE @pageIndex INT = 3;
DECLARE @pageSize INT = 4;

SELECT TOP (@pageSize) *

    SELECT ROW_NUMBER() OVER (ORDER BY 学号, 课程号) AS rownumber, *
    FROM StuCourse
```

#### 5. while 控制语句

```sql
BEGIN TRAN;

DECLARE @sumcredit INT;

SELECT @sumcredit = SUM(学分)
FROM Course;

WHILE @sumcredit < 50
BEGIN
    UPDATE Course
    SET 学分 = 学分 + 1;

    SELECT @sumcredit = SUM(学分)
    FROM Course;
END

SELECT *
FROM Course;

PRINT N'当前总学分为：' + CAST(@sumcredit AS VARCHAR(20));

ROLLBACK TRAN;
```

- 当前库起始总学分是 `21`，所以循环一定会执行。
- 为了写实验报告更安全，这里用事务包住，并在最后 `ROLLBACK`，避免把数据库永久改乱。
- 截图建议：先截 `Course` 原始数据，再截执行后结果和输出信息。

#### 6. 标量函数：按学号求总分

```sql
IF OBJECT_ID('dbo.fn_total_grade', 'FN') IS NOT NULL
    DROP FUNCTION dbo.fn_total_grade;
GO

CREATE FUNCTION dbo.fn_total_grade(@stu_no CHAR(6))
RETURNS INT
AS
BEGIN
    DECLARE @total INT;

    SELECT @total = SUM(CAST(成绩 AS INT))
    FROM StuCourse
    WHERE 学号 = @stu_no;

    RETURN @total;
END;
GO

SELECT dbo.fn_total_grade('070101') AS 总分;
GO

DECLARE @r INT;
EXEC @r = dbo.fn_total_grade '070101';
PRINT N'总分为：' + CAST(@r AS VARCHAR(20));
```

- 当前 `070101` 的总分是 `263`。
- 其组成是 `85 + 78 + 100`。

#### 7. 无参数存储过程：查询不及格学生

```sql
IF OBJECT_ID('dbo.proc_fail_info', 'P') IS NOT NULL
    DROP PROCEDURE dbo.proc_fail_info;
GO

CREATE PROCEDURE dbo.proc_fail_info
AS
BEGIN
    SELECT s.学号, s.姓名, s.专业名, c.课程号, c.课程名, sc.成绩
    FROM Student s
    JOIN StuCourse sc ON s.学号 = sc.学号
    JOIN Course c ON sc.课程号 = c.课程号
    WHERE sc.成绩 < 60;
END;
GO

EXEC dbo.proc_fail_info;
```
