# Agent Long-Term Memory Review

## What Went Well

- 这次调研把“论文线”和“开源工程线”同时拉通了，没有只停在概念层。
- 已经能把主流路线区分成几种清晰架构，而不是把所有项目都看成“memory toolkit”。
- 也把“遗忘曲线”从教育场景延伸到了 agent memory 的调度视角。

## What Was Weak

- 还没有系统整理 benchmark 和评测口径。
- 一些项目的性能数字主要来自项目方自述，第三方对照不足。
- 还没有把这些结论下沉成自己的 schema、表设计或 prompt 规范。

## Which Sources Were Most Valuable

- `MemGPT`
- `MemoryBank`
- `A-MEM`
- `RMM`
- `Mem0`
- `Graphiti/Zep`
- `LangMem`

## What To Do Differently Next Time

- 下一轮先做 benchmark 和 architecture matrix，再继续扩展项目列表。
- 把“可直接复用的工程模式”单独抽到一页，不和综述混在一起。
- 找一个最小原型做实测，减少只看 README 和论文的偏差。
